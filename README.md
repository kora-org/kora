# FaruOS
A simple Unix-like operating system written in Zig

## Componsnts

### [Lara](src/lara)
A hybrid kernel.

### [Xeptoboot](https://github.com/leap0x7b/xeptoboot)
A bootloader. (incomplete, Lara currently uses Limine)

### Koete
A GUI library.
